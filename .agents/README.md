# .agents

Дополнительные инструкции для агентов, работающих над `gly`.

Файлы:

- `implementation-rules.md` - практические правила реализации PowerShell-модуля.
- `validation-checklist.md` - минимальные проверки перед сдачей изменений.
- `handoff-template.md` - шаблон краткого отчета после агентской итерации.

Основной пошаговый план находится в `docs/agent-implementation-plan.md`.
