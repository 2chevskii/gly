# Renderer Commands

## Show-Gly

```powershell
Show-Gly -Path .
Get-ChildItem . | Show-Gly
```

Alias: `gly`.

## Show-GlyTree

```powershell
Show-GlyTree -Path . -Depth 2
```

Alias: `glytr`.

## Show-GlyGrid

```powershell
Show-GlyGrid -Path .
```

Alias: `glygr`.

Renderer commands apply active theme and glyph-set configuration. They are intended for interactive display.
