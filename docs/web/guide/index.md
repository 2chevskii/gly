# Guide

`gly` adds visual formatting to PowerShell file system output while keeping the underlying objects unchanged.

The MVP focuses on:

- PowerShell `7.0+`;
- standard table view customization through format data;
- glyph and color rendering for the `Name` column;
- session-only configuration;
- built-in and custom themes;
- built-in and custom glyph sets;
- explicit renderer commands for list, tree, and grid layouts.

Start with [Installation](installation.md), then follow the [Quick Start](quick-start.md).
